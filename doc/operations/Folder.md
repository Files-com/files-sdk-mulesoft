# Folder



## List Folders by Path

### XML Element

```xml
<filescom:list-folders />
```

### Parameters

| Parameter | Description |
| --------- | ----------- |
| path<br/>*String*<br/>**Required** | Path to operate on. |


### Example Operation

```xml
<filescom:list-folders config-ref="FilesCom" path="path" />
```
